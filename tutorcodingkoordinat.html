Tutorial: Membuat & Menghasilkan Gambar Koordinat dengan HTML & AI

Panduan ini akan membantu kamu memahami cara membuat kode HTML (misalnya dari AI) dan melihat hasilnya di browser web.

Bagian 1: Meminta Kode ke AI (Contoh Prompt)

Langkah pertama adalah mendapatkan kode HTML. Kamu bisa meminta bantuan AI untuk membuatkannya. Berikut adalah contoh cara meminta kode yang jelas:

Contoh Prompt untuk AI:

Tolong buatkan saya kodingan HTML untuk menampilkan dua garis lurus yang saling berpotongan. Jangan lupa sertakan label koordinat untuk setiap titik ujung garis dan juga di titik perpotongan kedua garis tersebut. Buatlah garisnya memiliki warna yang berbeda agar mudah dibedakan.

Setelah kamu memberikan prompt ini, AI akan memberikan kode HTML, kira-kira seperti ini (ini hanya contoh, kode sebenarnya bisa bervariasi):

<!DOCTYPE html>
<html>
<head>
<title>Garis Berpotongan dengan Koordinat</title>
<style>
  body {
    background-color: #f0f0f0;
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
    margin: 0;
    font-family: Arial, sans-serif;
  }
  .container {
    border: 1px solid #ccc;
    padding: 20px;
    background-color: white;
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
    text-align: center;
  }
  h2 {
    color: #333;
  }
</style>
</head>
<body>

<div class="container">
  <h2>Garis Berpotongan dengan Koordinat SVG</h2>
  <svg width="400" height="300">
    <line x1="50" y1="250" x2="350" y2="50" 
          style="stroke:red;stroke-width:2" />
    <text x="50" y="265" font-family="Verdana" font-size="12" fill="red">A (50, 250)</text>
    <text x="350" y="45" font-family="Verdana" font-size="12" fill="red">B (350, 50)</text>

    <line x1="50" y1="50" x2="350" y2="250" 
          style="stroke:blue;stroke-width:2" />
    <text x="50" y="45" font-family="Verdana" font-size="12" fill="blue">C (50, 50)</text>
    <text x="350" y="265" font-family="Verdana" font-size="12" fill="blue">D (350, 250)</text>

    <circle cx="200" cy="150" r="5" fill="green" />
    <text x="200" y="140" font-family="Verdana" font-size="14" fill="green" text-anchor="middle">Titik Potong (200, 150)</text>

  </svg>
</div>

</body>
</html>

Langkah selanjutnya adalah menyalin (copy) seluruh kode HTML ini.

Bagian 2: Menjalankan Kode dan Mengambil Screenshot

Ada dua cara utama untuk menjalankan kode HTML ini, tergantung apakah kamu menggunakan laptop dengan aplikasi pengolah kode (seperti VS Code) atau hanya menggunakan browser.

Opsi A: Menggunakan Laptop (dengan VS Code)

  1. Buka VS Code: Jika kamu belum punya, unduh dan instal dari code.visualstudio.com. Setelah terbuka, klik File > New File untuk membuat file baru.
  2. Tempel (Paste) Kode: Tempelkan seluruh kode HTML yang sudah kamu salin dari AI ke dalam file baru di VS Code.
  3. Simpan File sebagai HTML:
    • Klik File > Save As...
    • Berikan nama file, contoh: garis_potong.html. Penting: Pastikan nama file diakhiri dengan .html.
    • Pilih lokasi penyimpanan di komputer kamu dan klik Save.
  4. Buka di Browser:
    • Pergi ke folder tempat kamu menyimpan file garis_potong.html.
    • Klik ganda pada file tersebut.
    • Secara otomatis, file akan terbuka di browser web default kamu (Chrome, Firefox, dll.), menampilkan hasil dari kode HTML-mu.
  5. Ambil Screenshot: Setelah hasilnya muncul di browser, ambil screenshot.
    • Windows: Tekan PrtSc (Print Screen) atau Windows Key + Shift + S.
    • macOS: Tekan Cmd + Shift + 3 (seluruh layar) atau Cmd + Shift + 4 (area tertentu).
    • Linux: Gunakan fitur screenshot bawaan sistem operasi.

Opsi B: Menggunakan Browser (Tanpa Laptop, Contoh: HP/Tablet dengan OneCompiler)

  1. Buka OneCompiler: Buka browser web kamu (misalnya Chrome atau Safari) dan kunjungi alamat: https://onecompiler.com/html/
  2. Tempel (Paste) Kode:
    • Di halaman OneCompiler, kamu akan melihat area teks di sebelah kiri atau atas.
    • Hapus kode contoh yang mungkin sudah ada di sana.
    • Tempelkan seluruh kode HTML yang sudah kamu salin dari AI ke area teks tersebut.
  3. Jalankan (Run) Kode:
    • Cari tombol "Run" (biasanya di bagian atas atau bawah area kode).
    • Klik tombol "Run" tersebut.
  4. Lihat Hasil:
    • Hasil dari kode HTML kamu akan ditampilkan di area pratinjau yang ada di halaman OneCompiler.
  5. Ambil Screenshot: Setelah hasilnya terlihat di OneCompiler, ambil screenshot.
    • Android/iOS: Biasanya dengan menekan kombinasi tombol Power + Volume Bawah (atau Power + Tombol Home, tergantung model perangkatmu).

PENTING: Pengumpulan Screenshot

Setelah kamu berhasil mengambil screenshot dari hasil gambar koordinat, kamu bisa mengunggahnya biasanya melalui classroom yang Bapak Aja kirimkan.
